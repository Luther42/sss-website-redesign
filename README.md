## Project Overview

A full UI/UX redesign of the **Social Security System (SSS)** website as an
academic mockup project. The goal was to modernize the user experience while
staying true to the official SSS brand identity.

### What was redesigned:
- Simplified navigation and overall layout for everyday users
- Improved readability and visual accessibility throughout the site
- Added icons to interactive buttons to improve engagement and clarity
- Overhauled the carousel component for a smoother experience
- Maintained strict adherence to the official SSS color palette
- Redesigned multiple pages including Home, Membership, Loans, Benefits, and more
